## About

World clock, alarm, stopwatch, and timer in one application.

## Features

- World clock
- Multiple alarms
- Stopwatch with laps
- Countdown timer

## Installation

### From App Store
Open the **App Store** in zOS and search for "Clock".

### Via npm
```bash
npm install @zos-apps/clock
```

## Usage

- Add cities to world clock
- Set alarms with + button
- Use tabs to switch modes



## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
